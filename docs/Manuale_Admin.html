Sezione 1

Ruolo e responsabilità dell'Admin

Cosa distingue l'Admin dal Manager e quali aree di sistema gestisce.

L'Admin ha accesso completo a tutte le aree dell'azienda e a funzionalità di sistema riservate. Le responsabilità principali sono:

Gestire il calendario festività per tutti gli anniSistema

ℹ️ Funzionalità condivise
L'Admin ha anche accesso a tutte le funzionalità del Manager (assegnare turni, export, statistiche, impostazioni area). Per quelle funzionalità consulta il Manuale Manager.

Sezione 3

Gestione sistema — Template Excel

Carica e gestisci il template aziendale usato per l'export dei turni.

Vai su Sistema nel menu (visibile solo all'Admin). I manager che tentano di accedere vengono reindirizzati.

📷
Pagina Sistema — sezione template
Inserire qui lo screenshot della pagina /admin/sistema

Caricare un nuovo template

1
Prepara il file .xlsx

Il template deve avere il formato aziendale corretto (struttura colonne predefinita). Il sistema compilerà solo i dati dei turni, lasciando intatti logo, firme e intestazioni.

2
Carica il file

Trascina il file .xlsx nella zona di upload o clicca per selezionarlo.

3
Conferma

Il template viene salvato nello storage e diventa immediatamente disponibile per i manager.

❌ Template mancante
Se nessun template è caricato, i manager vedono un errore quando tentano di generare l'Excel. Carica sempre un template prima di consentire l'export.

Puoi vedere la lista dei template caricati con nome, dimensione e data. Il template più recente viene usato automaticamente per tutti gli export.

Sezione 4

Calendario festività

Gestisci le festività che il sistema tratta come giorni speciali per la reperibilità.

Nella pagina Sistema trovi anche il calendario festività. Le festività influenzano l'assegnazione turni (peso ×3 nello score equità se obbligatorie) e la visualizzazione nel calendario dei dipendenti.

📷
Sezione festività — lista per anno
Inserire qui lo screenshot della sezione festività con anni collassabili

4.1 Importare festività automaticamente

1
Seleziona l'anno

Usa il campo anno (es. 2026, 2027).

2
Clicca "Importa festività"

Il sistema scarica automaticamente le festività ufficiali italiane per quell'anno.

3
Le festività appaiono nella lista

Le festività già presenti non vengono duplicate.

4.2 Aggiungere una festività manuale

Per festività aziendali o locali non presenti nel calendario ufficiale:

1
Inserisci data e nome

Es. data: 15/06/2026, nome: "Sagra del paese".

2
Scegli se è obbligatoria

Spunta "Festività obbligatoria" se deve pesare ×3 nello score equità. Altrimenti viene trattata come un giorno speciale normale.

3
Clicca "Aggiungi"

La festività è immediatamente attiva nel sistema.

4.3 Gestire le festività esistenti

  • Toggle obbligatoria / non obbligatoria — modifica il peso nello score equità
  • Elimina — clicca la X per rimuovere. Non è possibile modificare una festività: eliminala e ricreala.
⚠️ Turni già assegnati
Modificare o eliminare una festività non rimuove i turni già assegnati in quel giorno. Agisci prima dell'assegnazione.
Sezione 5

Importazione storico turni

Carica turni storici da file Excel per popolare il database con i dati pregressi.

Nella pagina Sistema trovi la sezione "Importa storico reperibilità". Serve per migrare dati da sistemi precedenti o per caricare turni passati che non erano ancora nel database.

📷
Sezione importazione storico
Inserire qui lo screenshot della sezione importa storico

Come funziona

1
Prepara il file Excel

Il file deve avere il formato identico al template aziendale (stesso usato per l'export). Il sistema legge il mese dalla cella A3 e i cognomi dalle colonne D/E, righe 10-40.

2
Carica il file

Trascina il file Excel nella zona di upload.

3
Risolvi le ambiguità

Se il sistema non riesce ad abbinare un cognome a un dipendente, compare un dialog per la risoluzione manuale. Puoi scegliere il dipendente corretto dalla lista.

4
Conferma l'importazione

I turni vengono inseriti. I mesi passati diventano confirmed, il mese corrente locked.

Dialog risoluzione ambiguità cognomi

⚠️ Caricamenti multipli
Puoi caricare più file nella stessa sessione (anche con gli stessi nomi) in modo sequenziale. Il sistema gestisce i duplicati automaticamente.
Sezione 6

Gestione utenti

Crea e gestisci tutti gli utenti del sistema: dipendenti, manager e admin.

Vai su Utenti nel menu. L'Admin vede tutti gli utenti di tutte le aree, con la possibilità di filtrare per area.

📷
Lista utenti con filtro area
Inserire qui lo screenshot della pagina /admin/utenti con dropdown area

Azioni disponibili (solo Admin)

Cambiare il ruolo di un utente

Usa il dropdown accanto al nome per cambiare il ruolo: Dipendente → Manager o viceversa. Il cambio viene applicato immediatamente. Se promuovi un utente a manager di un'area, il campo manager di quella area viene aggiornato automaticamente.

Filtrare per area

Usa il dropdown "Area" in cima alla lista per visualizzare solo gli utenti di una specifica area.

Attivare / Disattivare

Come per il manager, il toggle attiva/disattiva l'accesso. Un utente disattivato non può fare login.

Eliminare un utente

Clicca la X e conferma. L'eliminazione rimuove l'account e tutti i dati correlati (turni, disponibilità). Operazione irreversibile.

❌ Restrizioni di sicurezza
Non puoi cambiare il tuo stesso ruolo, né disattivarti o eliminarti. Non puoi promuovere un manager ad admin tramite l'interfaccia utenti — i ruoli admin vengono gestiti direttamente nel database.
Sezione 7

Panoramica equità multi-area

Monitora la distribuzione dei turni su tutte le aree dell'azienda.

Vai su Equità nel menu. Questa pagina è esclusiva dell'Admin e mostra una panoramica di tutte le aree.

📷
Pagina equità multi-area
Inserire qui lo screenshot della pagina /admin/equita

Cosa vedi

Per ogni area è mostrato:

  • Nome area e numero dipendenti
  • Score medio dell'area
  • Score minimo e massimo
  • Delta (differenza tra max e min — indica quanto è equa la distribuzione)

Un delta basso indica distribuzione equa. Un delta alto segnala che alcuni dipendenti hanno molti più turni di altri.

Dettaglio per area

Clicca su un'area per espandere il dettaglio con il ranking completo dei dipendenti di quell'area.

Analisi storica

Usa i selettori mese/anno per analizzare mesi precedenti.

Sezione 8

Sblocco mesi confermati

Operazione riservata all'Admin per correggere errori post-invio email.

Solo l'Admin può sbloccare un mese in stato confirmed. Questa operazione è necessaria quando il manager ha inviato l'email con turni errati e deve correggere e re-inviare.

❌ Attenzione — operazione delicata
Sbloccare un mese confirmed azzera lo stato "Email inviata". Se l'email era già stata recapitata ai dipendenti, dopo le correzioni sarà necessario re-inviare manualmente. Informare sempre il manager prima di procedere.

Come sbloccare

1
Vai su Disponibilità

Menu laterale → Disponibilità.

2
Seleziona l'area

Usa il selettore area in cima alla pagina per scegliere l'area del mese da sbloccare.

3
Naviga al mese corretto

Usa le frecce ← → per arrivare al mese confermato.

4
Clicca "Sblocca"

Il pulsante "Sblocca" appare solo per i mesi confirmed. Comparirà un dialog di conferma.

5
Informa il manager

Il mese torna a stato open. Il manager può ora correggere i turni, ri-bloccare e re-inviare.
